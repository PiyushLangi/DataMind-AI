# 🚀 DataMind AI

DataMind AI is an AI-powered automated data analytics and machine learning platform inspired by tools like Power BI, Tableau, and ChatGPT.

It allows users to upload datasets, preprocess data, build interactive dashboards, generate visualizations, interact with AI, and create machine learning workflows — all inside a modern glassmorphism UI.

---

# ✨ Features

## 📂 Dataset Upload
- Upload CSV datasets
- Automatic parsing and column detection
- Dataset preview and metadata extraction

## 🧹 Data Preprocessing
- Missing value handling
- Data cleaning workflow
- Column analysis
- Dataset preparation pipeline

## 🎯 Target Column Selection
- Intelligent target variable selection
- ML problem type detection
- Classification / Regression workflow

## 🤖 AI Chat Assistant
- Gemini-powered AI assistant
- Dataset-aware conversations
- AI-generated insights and recommendations

## 📊 Advanced Visualization Builder
- Power BI inspired dashboard builder
- Drag-and-drop chart system
- Interactive visualizations
- Cross-filtering engine
- Slicers and filters
- Analytics panel

## 🧠 Machine Learning Workflow
- ML model selection
- Problem type handling
- Dataset analysis pipeline

## 📑 Reports
- Export analytics reports
- Generate dashboard summaries
- AI insight reporting
